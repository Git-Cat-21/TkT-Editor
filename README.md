# TkT~Ed
---------------
### Usage :

```bash
git clone https://github.com/Git-Cat-21/TkT-Editor.git
cd TkT-Editor
pip install -r requirements.txt
python main.py
```

-   The TkT Editor window should now open. Explore the TkT Editor and enjoy its features
